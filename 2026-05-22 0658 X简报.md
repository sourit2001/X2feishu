### 🧠 @emollick｜AI 的“阶级固化”：Chatbot 人人可得，Agent 富人专属

**主要内容**  
Ethan Mollick 连续发布两条推文，直击当前 AI 行业的结构性矛盾：
- 我们严重缺乏算力，导致复杂 Agent 工作流的 Token 成本飙升。
- 单轮对话的 Chatbot 越来越便宜（甚至免费），但复杂 Agent 要消耗数千倍 Token，只有最有钱的公司和最高优先级的场景才能承受。

**核心观点**  
AI 的“民主化”只是表象。真正能完成复杂工作的 Agent 将成为稀缺资源，加剧技术使用的阶层分化。普通用户可能被困在“好用的聊天机器人”里，而大厂依靠算力护城河垄断 Agent 红利。

### 🧑‍💻 @xiaohu｜两大实用更新：ChatGPT 插件直连 PowerPoint，Codex 锁屏干活

**主要内容**  
小虎带来了两条重磅工具更新：

**1. OpenAI 官方“ChatGPT for PowerPoint”插件**  
- 安装后，PPT 右侧可出现 ChatGPT 对话框，支持：
  - 从零生成新 PPT（用大白话指令）
  - 改进已有 PPT（读取并分析）
  - 将会议记录、数据表整理成可展示内容
  - 总结 PPT 故事线
  - 🎯 输出内容在 PPT 中**完全可编辑**，不是死图。

**2. Codex 推出 Locked Use（锁屏使用）功能**  
- 允许在 Mac 锁屏后继续操作电脑完成自动化任务。
- 可通过手机远程给 Codex 下达指令，锁屏状态工作。
- 底层采用苹果官方 Authorization Plug-in 机制，临时解锁 → 干活 → 自动锁屏。
- 全程屏幕保持黑屏，保护隐私。

**核心观点**  
这两项更新标志着 AI 正在融入主流生产力工具（Office）和高级自动化场景。锁屏工作功能进一步降低了无人值守自动化的门槛，使 AI Agent 能力更接近“24 小时数字员工”。

**实用工具/资源**  
- ChatGPT for PowerPoint 安装方法：PowerPoint → Home → Add-ins → 搜索“ChatGPT”；或 Microsoft Marketplace 下载 [🔗 链接](https://t.co/CcJgwnxxNW)  
- Codex Locked Use 设置：Codex → Computer Use → 打开控制力 [🔗 链接](https://t.co/FUukJOPWXI)

## 🔥 综合启发

1. **算力决定 Agent 门槛，别被“免费 Chatbot”迷惑**  
   - Ethan Mollick 的观察提醒我们：如果你正在开发需要多步推理、长期记忆、循环调用的 Agent 产品，务必提前规划 Token 预算。否则可能在商业化时被算力成本卡住脖子。  
   - 对于普通用户，免费 Chatbot 足够做日常助手，但高级自动化任务迟早要付费。

2. **Office 集成 AI 的窗口已打开，尽早优化工作流**  
   - ChatGPT for PowerPoint 这类原生插件正在改写办公软件的使用方式。建议至少尝试一次，体验“一句话生成可编辑 PPT”+“AI 总结会议纪要”等场景，避免被同行拉开效率差距。  
   - 同样值得关注的是 Codex 的锁屏自动化——如果你有大量后台脚本、文件处理、网页操作等重复劳动，它可以成为你的“夜间数字员工”。

3. **图像编辑领域迎来效率革命，BFL 等新秀值得跟进**  
   - Black Forest Labs 的 inpainting 测试结果令人印象深刻。如果你从事设计、电商、内容修改等需频繁处理图片的工作，这类工具能大幅缩短修图时间。  
   - 对于休闲用户，像 icreatelife 分享的趣味 Prompt 也提醒我们——AI 创造力不应只限于“解决工作”，探索其娱乐价值同样能带来启发。
