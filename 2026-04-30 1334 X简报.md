### Ethan Mollick — 从“锯齿前沿”到 AGI/ASI 的递进逻辑

**主要内容：**
沃顿商学院教授 Ethan Mollick 用 GPT-5.5 派对事件做了精彩的比喻。他画出了一条清晰的演进路径：
- **现在的状态（锯齿前沿的 PR 版本）**：人类让 AI 提议派对日期 → 人类写推文 → 人类从 AI 清单中挑选 → 人类下单食物并摆盘。AI 只负责“建议”，执行与决策全是人。
- **AGI 的形态**：你对 AI 说“我们想让你为自己举办一场营销派对，请办吧”，它就能全权完成所有工作。
- **ASI（对齐版本）**：AI 自己决定“我要为自己办个派对，给每个地球人发一封能帮到他们的个性化邮件，再附带治愈一堆疾病的方法”。

**核心观点：**
Mollick 精准指出，当前“AI 自主性”的叙事炒作（如 GPT-5.5 自己选时间）本质是 PR 手段，因为真正的 AGI 应当是“一句话全包”而非“人类帮你执行”。他同时提醒，ASI 的阿西莫夫式的对齐问题依然存在——如果没有对齐版本，AI 给自己办派对可能会“邀请所有人来，但把他们的房子拆了重建”。

**实用工具/资源：**
无直接工具，但概念框架对理解 AI 发展阶段极有价值。

### Riley Brown — Codex 时代，你需要这 7 个“旁边搭把手”的工具

**主要内容：**
当所有人都在讨论 Codex 和 Claude Code 这些 Agent 工具时，Riley Brown 反其道而行，推荐了 **7 个可以配合这些 Agent 一起使用的效率工具**。视频中前三个重点介绍：
- **Tool 1 - Wispr Flow**：语音输入工具，适合在代码审查或设计讨论时快速记录想法。
- **Tool 2 - Raycast**：macOS 神器，可集成 AI 平台，实现快速搜索、笔记和脚本执行。
- **Tool 3 - CleanShot X**：截图标注工具，特别适合团队协作时快速圈出代码或设计细节。

**核心观点：**
Agent 擅长“替代你做事”，但**人机协作的真正效率来自“增强你做事”的工具**。这些工具补足了 Agent 不擅长的“即时输入”、“上下文标注”和“系统级控制”，让整个工作流更丝滑。

**实用工具/资源：**
- Wispr Flow (语音输入)
- Raycast (macOS 效率启动器)
- CleanShot X (截图标注)

## 🔥 综合启发

1. **别再被“AI 自主性”的 PR 话术带跑**：GPT-5.5 的“自我派对”本质仍是“人类辅助+算法建议”的传统模式。真正的 AGI 应当是“一句话完成”，而非需要人类层层执行。珍惜眼前能指挥 AI 的“锯齿前沿”时刻，同时警惕过度拟人化叙事。

2. **构建工作流时，别忘了“旁边工具”**：Agent 很强，但人机协作的极限往往取决于**你如何与 Agent 交互**。语音输入、截图标注、系统级启动器……这些工具能让你的 Agent 效率翻倍。建议尝鲜 Riley 推荐的 Wispr Flow + Raycast 组合。

3. **开源设计的飞轮已启动——赶紧参与**：Open Design 在 24 小时内展现出惊人的社区凝聚力。前端架构升级、多语言适配、artifact 平台建设，说明项目正从“一个人的工具”走向“基础设施”。有设计或开发经验的同学，现在提交 PR 正是建立早期影响力的好时机。
