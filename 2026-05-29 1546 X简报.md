## 👤 博主深度速递

### 🧠 归藏 (@op7418) —— 动态工作流：比模型升级更具革命性的“生产力引擎”

归藏今天的信息量极大，其对 Claude 生态的解读尤其一针见血。

- **主要内容：**
  1. **Opus 4.8 惊喜发布**：归藏第一时间捕捉到 Claude Opus 4.8 的发布。他明确指出，相较于 4.7，该版本在所有能力上均有提升。其中最关键的进步是**模型自我审查代码的能力**——以前让模型自己 review 代码基本无效，现在则有显著改观。同时，Fast 模式价格大幅下调至原价的三分之一。
  2. **“动态工作流”意义更重大**：归藏强调，相比于模型本身的发布，昨晚推出的“动态工作流” (Dynamic Workflows) 具有更深远的意义。它延续了并发 Subagent 逻辑，但突破了数量限制，可启动**数百个 Subagent**，协同完成极其庞大和复杂的任务，例如：全面审计整个代码库，或快速收集信息生成一份庞大的调研报告。
  3. **“藏师傅”Skills 实战出圈**：观察到已有多个博主利用他分享的“PPT Skills”和“小红书配图 Skills”，在小红书上创作出了千赞级的爆款内容，证实了这些技能的实用性。

- **核心观点：**
  - 模型能力的微调固然是好事，但**系统架构层面的创新**（如动态工作流）对生产力的提升更具决定性。
  - 好工具必须结合好的应用场景和“技能”（Skills），才能真正转化为内容创作者的竞争壁垒。

- **实用工具/资源：**
  - **Claude Opus 4.8**：新发布的模型，具备更强自我审查能力。
  - **Dynamic Workflows**：Claude Code 中的新型多智能体工作流。

### 🧑‍💻 宝玉 (@dotey) —— 冷静的实用主义者与“漏洞猎人”

宝玉的视角更偏向一线开发者的实战体验，对细节和资源非常敏感。

- **主要内容：**
  1. **写作能力对比**：宝玉提供了宝贵的亲测体验：“写作4.6肯定比4.7强，是不是比4.8强还在测试中，目前感觉还是4.6好。” 这表明模型的能力并非线性提升，不同任务仍需选择最合适的版本。
  2. **Opus 4.8 官方技术细节**：宝玉转发了针对 Opus 4.8 Model Card 的解读，内容包括：1M 上下文 GraphWalks 带来的显著提升、大量 multi-agent 内容、模型更“诚实”地承认自身代码缺陷、在 Agent 安全测试中出现倒退等关键信息。
  3. **Codex 额度预测**：他预测 Codex 明日会重置额度，暗示了该工具的火爆程度远超预期。
  4. **动态工作流“开源平替”**：宝玉转发了 `pi` 版本的动态工作流，该版本支持使用任意大模型（如 DeepSeek, Codex-5.5）来复现 Opus 4.8 的动态工作流功能。作者正是此前破解 Claude Code 的 Generative UI 的大神。

- **核心观点：**
  - 模型选择不能唯“版本号”论，即使是最新的模型，在特定领域（如写作）也可能不如旧版本。开发者应保持灵活的模型切换思维。
  - 开源社区的反应速度极快，官方功能发布后几小时内，基于不同模型的“平替”方案就已经出现，极大地降低了创新门槛。

- **实用工具/资源：**
  - **pi 版本的动态工作流**：支持任意大模型的开源实现。
  - **Opus 4.8 Model Card 解读**：获取详细的性能优劣信息。 (转自 @dongxi_nlp)

### 🎓 Ethan Mollick (@emollick) —— 研究型应用与能力深度评估

Ethan 教授从学术和研究的角度，为我们揭示了顶级模型在复杂任务上的惊人实力。

- **主要内容：**
  1. **GPT-5 Pro 系列的单挑之王**：Ethan 指出一个有趣且值得注意的现象：从去年夏天开始，GPT-5 Pro 系列模型在最具挑战性的单次生成任务上一直保持着最佳水准，长期以来鲜有对手。
  2. **GPT-5.5 Pro 的学术擅场**：他强调 GPT-5.5 Pro 是目前生产许多新颖数学证明的模型，也是你**审查技术或学术论文缺陷时**应该使用的模型。
  3. **AI 的“经济学家”角色扮演**：他分享了一个案例，Claude 在模拟经济学家完成一篇论文的稳健性检验后，给出了非常人性化的评价：“在 1-10 分的识别强度上，我现在会打 4.5 分...比以前好，但还差得远。” 这展现了 AI 处理复杂推理和“元认知”的潜力。
  4. **人类历史可视化**：他让 Claude Code 中的 Opus 4.8 将所有曾经存在过的人类进行可视化，认为这是测试 AI 结合研究、代码、设计和统计能力的有趣方式。

- **核心观点：**
  - 对于需要极高精度和“一次命中”的“硬问题”，GPT-5 Pro 系列仍是当前的最优解。
  - AI 不仅能执行任务，还能对其产出进行“自我批评”和量化评估，这标志着 AI 辅助研究进入了更成熟的阶段。

- **实用工具/资源：**
  - **GPT-5 Pro / GPT-5.5 Pro**：当前处理高难度、高精度任务的首选模型。
  - **Opus 4.8**：可用于结合多种能力进行复杂创作和可视化。
