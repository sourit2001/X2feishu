### @小虎

Apodex 是一套“自我进化的重载求解器”，专门啃那些没有现成答案、需要大量调研的硬骨头。它最多可派出 150 个子 Agent 并行探索，累计跑 15,000 步，在 BrowseComp 上碾压 GPT-5.5-pro，在 DeepSearchQA 上超过 Claude-Opus-4.8 和 Kimi-K2.6。小虎重点关注其 Heavy-duty 模式：开启 Agent 团队+全局验证后，BrowseComp 从 75.5 暴涨到 90.3，FrontierScience-Research 提升 18.4 个百分点。更狠的是，Apodex 开源了 35B（A3B）的小模型系列，其中 4B-SFT 仅用深度研究 SFT 数据就能超越所有开源 30B 级模型。这套框架已经在 FutureX 预测竞赛中连续两周包揽前四名。此外小虎还点了一句：Google 花 27 亿美金挖来的人，不到两年跳槽 OpenAI——大企业病不治，Gemini 短暂领先后就掉队了。

### @seclink

他正在用三句话教 AI 如何重构项目知识体系。第一，让 AI 根据当前项目全链路，按 Google Open Knowledge Format 规范在根目录创建 `docs/okf/`，为每个核心数据库表和 API 生成独立的 Markdown 概念文件，并用 YAML frontmatter 标注类型。第二，让 AI 编写一个 Python 解析器 `OKFReader`，扫描目录、解析 YAML 元数据和 Markdown 内容。第三，把这些规则写进 `CLAUDE.md`，使得后续提问时 AI 自动知道所有业务定义和指标逻辑都保存在 `okf/` 下。这不是简单的代码生成，而是一套“知识工程自动化”的范式：用 AI 自我维护团队知识库，让结构化元数据成为模型与企业数据的桥接层。

### 本期趋势

本时间段内三条暗线浮出水面：AI 治理的地缘政治分歧正在从专家讨论进入闭门谈判层面，Anthropic 与 Google DeepMind 联手推动“美国主导的盟友联盟”框架，但法国、加拿大等国要求多边而非单边。同时，量子计算和光电子供应链出现明确落地信号——第二套 ROSIE 系统即将交付，激光器公司正在向光模块/光引擎系统商跃迁。此外，开源深度研究模型（Apodex-1.0-mini）以 4B 参数量级碾压 30B 模型，预示小模型+Agent 协作可能替代大模型的 brute-force 路线。三件事共同指向：AI 和硬科技的规则制定权与实物供应链正在同步收紧。
