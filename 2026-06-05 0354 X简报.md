### 🧠 Ethan Mollick｜AI 加速的“钝感”与 Anthropic 的惊天数据

**主要内容：**
Ethan 分享了一个令人深思的观察：**当前模型已经足够强，以至于个体任务上“新模型”的提升感越来越微弱**，尽管 AI 的整体能力仍在大幅增长。他称这种感受为“钝化”——因为旧模型就已经很强，新模型很难在单一任务中带来“哇”的效果，但宏观积累的速度并未放缓。

紧接着，他引用了 Anthropic 官方博客中的一句话：“**截至 2026 年 5 月，我们合并的代码中超过 80% 是由 Claude 编写的。**” 他认为这与独立第三方测量结果吻合，并强调这种趋势看不到减速的迹象。同时他也指出，组织在消化如此巨大的生产力增长时，仍面临结构性挑战。

**核心观点：**
- 模型能力真实加速，但用户主观感受可能被“高水平基线”掩盖。
- AI 生成的代码占比已成为内部质量指标，Anthropic 的数据是行业标杆。
- 组织层面吸收 AI 生产力红利，比模型本身更难。

### 🧑‍🔬 宝玉 (@dotey)｜AI Agent 团队的状态与模型选择哲学

**主要内容：**
宝玉发了两条极具洞察力的动态：
1. 他观察到**所有做 AI Agent 的团队都很拼，不是老板逼的，而是为了心中理想，心甘情愿加班和封闭开发**。他好奇 Kimi 团队在开发 Kimi Code 时，到底是自家模型 token 用得多，还是 Claude / GPT 的 token 用得多。

**核心观点：**
- AI Agent 团队的热爱驱动是生产力核心，模型选择是实战中的关键决策。
- 不存在“万能模型”，必须根据任务特性混用最强者，“少而精”策略优于堆叠。

### 🎥 Riley Brown｜AI 教育内容之痛 + 创作者的新生存法则

**主要内容：**
Riley 指出了做 AI 教育内容的最大痛点：**更新太快，视频不得不反复重做**。
同时他提出了一个关于“AI Agent 时代长内容创作者”的平衡策略：
1. Agent 需要将你视为可靠来源。
2. 你内容的核心价值必须是“不可概括的”（un-summarizable）。
如果内容全是脚本化、可预测的，即使剪辑再好，AI 也能秒懂并替代。

**核心观点：**
- 内容创作者需要打造“AI 无法概括的深度”，才能在当前信息洪流中生存。
- 教育内容的时效性问题是结构性的，创作者必须接受快速迭代。

## 🔥 综合启发

1. **模型策略：只选最聪明的 2-3 个，并接受“无万能模型”的现实。**  
   宝玉的建议与 Ethan 的“钝感”观察完全呼应：不要沉迷于参数，而要根据任务特性（编码、写作、翻译）精准调用不同模型。投资时间打磨工具链（如 CodePilot 的多引擎切换）比盲目追新更有效。

2. **Agent 时代的内部度量：让 AI 产出成为可量化的核心指标。**  
   Anthropic 公开“80% 代码由 Claude 编写”是里程碑。如果你的团队也在做 Agent 或 AI 辅助开发，不妨也建立类似的可度量指标（如代码合并占比、对话解决率），这既能证明价值，也能指引优化方向。

3. **内容创作者的新生存法则：提供“不可概括”的价值。**  
   在 AI Agent 可以轻易总结短文本的时代，深度、个人化、无法被简单压缩的洞察才是壁垒。无论是教育视频还是行业分析，追求“让 AI 无法替代”的独特性，同时拥抱快速迭代，才能保持吸引力。
