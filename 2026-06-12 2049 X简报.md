### @Serenity

芯片供应链的摩擦点正在发热。Serenity 观察到 VPEC 对 Epiwafers 的最新涨价，这直接验证了 IQE 和 Landmark 等公司拥有强大的定价能力。一个有趣的信号是，MTSI 对 IQE 的投资是为了锁定产能，说明上游材料的节点正变得极难替代。而日本的 WF6 供应链因中国的出口管制被切断，影响波及 TSMC、三星和 SK Hynix，这已经超出了商业博弈的范畴。Serenity 认为，当下的价格异动更多是美国大型机构的布局信号，JP Morgan 对 SIVE 的持仓从 0.4% 猛增至 5.3%，Fidelity 等基金也已入局。在他看来，AI 霸权的竞争已经演变为一场围绕“战略胶点”的全球供应链战争，日本掌握的许多关键材料是其中的暗线。

### @TutureTom

Open Design 正在改变 UI 设计师的工具使用方式。0.10.0 版本新增了内建浏览器，这意味着设计师可以直接从一个窗口内完成截图并一键转换为设计稿。这与 Figma 的 Chrome 插件形成鲜明对比：后者需要用户在多个窗口间烦琐跳转。这次更新还收录了涵盖灵感、真实产品 UI、动效、配色、字体、图标以及设计系统的参考网站库。从项目开源仅 45 天便获得超过 63.6K GitHub Stars 来看，这款工具显然抓住了设计师对高效、沉浸式工作流的需求。

### 本期趋势

芯片产业正从“全球化效率”转向“地缘政治韧性”。上游材料的涨价和日本 WF6 供应链被切断，意味着只要卡住一个节点就能瘫痪整个 AI 芯片生产链，这令投资逻辑从利润转向供应链安全。与此同时，AI 工具正在深入微观工作流：God of Prompt 提供的是从文案到转化的全流程智能替代；Open Design 则试图成为设计师的唯一窗口。这些变革意味着去中介化和对传统工作流的深度重构正在加速。
