### 2️⃣ Ethan Mollick (@emollick) — Gemini Omni 重新定义视频 AI

#### 主要内容
Ethan 展示了 Gemini Omni 的惊人能力：直接对一段 1896 年的经典火车视频进行原生编辑，瞬间将其变成子弹头列车、乐高风格、加入时间旅行者、千足虫、布偶等元素，甚至保留镜面反射真实感。

#### 核心观点
- **“真·多模态” vs 胶带式多模态**：Gemini Omni 之所以不同，是因为它是 **完全多模态**——视频输入后，AI 能直接理解像素级语义并进行编辑，而非像其他工具那样依赖外部分割、帧序列再合成。
- **视频编辑门槛骤降**：以往需要 Premiere Pro + After Effects + 大量特效师工作的成果，现在通过自然语言提示即可在几秒内完成，且能保持物理一致性。

### 3️⃣ Riley Brown (@rileybrown) — Agent 自主构建并发布 App 的狂想

#### 主要内容
Riley 演示了一个闭环工作流：看到有趣的 App 想法 → 截图 → 通过 iMessage 发送给专属 Agent → Agent 直接生成 Swift 应用 → 返回链接 → 用户下载到手机。他预言：**6 个月内，每个 Agent 都能完美构建任何移动应用并能无缝发布到 App Store**。

#### 核心观点
- **自动化将吞噬 App 开发**：从想法到产品完全由 AI 自动执行，人类只负责“截图 + 一句话指令”。他甚至强调“对构建这个兴趣不大，因为这将是所有 Agent 的默认能力”。

## 🔥 综合启发

### 🚀 1. Agent 正在成为“全栈程序员”，尽快拥抱“截图即开发”范式
从 Riley Brown 的演示看，未来 6 个月内，任何 AI Agent 都可能自动完成从想法到上架的全流程。**建议**：现在就开始搭建自己的专属 Agent（例如利用大模型 API + 自动化脚本），尝试把日常小需求（如生成一个简易工具、响应式页面）交给它，积累对 Agent 能力的直觉，抓住生态位。

### 🎬 2. 多模态 AI 的“原生时代”开启，视频编辑和内容生产将爆发
Gemini Omni 证明了真正的多模态不是拼凑，而是内建理解。这预示着一波视频内容创作革命：个人可以用自然语言命令 AI 重拍经典电影、增加特效、修改物体。**建议**：运营者应立即尝试这类模型，将其用于短视频创作、素材快速迭代，用“一句话改视频”的能力大幅提升内容产出效率。

### 📈 3. Token 成本下降倒逼“规模化使用”，而细分场景会成为新蓝海
a16z 的数据显示智能总消耗已创新高。对于开发者来说，曾经昂贵的 AI 能力（如长视频分析、百万级 Token 上下文窗口）如今可以大方使用。**建议**：在项目早期就设计高 Token 消耗但高价值的场景（如全量数据审核、长篇内容摘要、实时多语言翻译），吃透“便宜智能”的红利，而不是纠结于成本优化。同时，关注 a16z 每周图表，及时调整基础设施策略。
