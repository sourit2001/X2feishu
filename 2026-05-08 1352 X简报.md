### 🧪 归藏 (@op7418) & 🐯 小虎 (@xiaohu) —— Codex 浏览器插件引爆效率革命

**主要内容：**
Codex 刚刚更新了 Chrome 浏览器扩展，这项功能堪称“数字员工”的雏形。它能够直接控制你已登录的 Chrome 浏览器，跨页面并行执行任务，且后台运行不干扰用户正常浏览。归藏实测发现，Codex 可以启动多个子 Agent 同时操作不同网页，并且会在浏览器中创建 Tab 分组，方便用户追踪每个 Agent 的动态。小虎进一步指出，Codex 利用的是你已经登录的账号状态，所以能直接操作内部 CRM、企业后台等需要身份验证的系统，全程无需用户提供密码。

**核心观点：**
1. Codex 正在从“代码生成工具”进化为“自动化工作流平台”，浏览器扩展是打通真实数字世界的关键入口。
2. 真正的杀手锏在于“并行”与“后台”——AI 可以像人类一样同时操作多个网页，但不会打断你当前的工作。
3. 安全风险不容忽视：用户需谨慎授权 Codex 操作敏感页面（如银行、支付系统）。

**主要内容：**
Google 突然发布了一款无屏幕的 Fitbit 运动手环，重量仅 5 克，续航长达一周，充电 5 分钟即可使用一整天。硬件售价 99 美元（一次性），可选每月 9.99 美元的订阅服务，比 Whoop 和 Oura 的性价比都高出不少。它能自动监测心率、心律（含房颤警报）、血氧、皮肤温度、睡眠分期、心率变异性等数据。真正的杀招在于：搭配 Gemini 驱动的 Google Health Coach，用户只需拍一张健身器械的照片，AI 就能生成定制训练计划。

### 🔒 seclink —— Next.js 严重安全漏洞 & GPT-Realtime-2 发布 & Gemini 智能体新能力

**主要内容：**
1. **安全警报：** Cloudflare 和 Next.js 团队联合披露了多个影响 React Server Components 和 Next.js 的严重安全漏洞，官方已火速发布 Next.js 16.2.6 和 15.5.18 修复版本。建议所有 Next.js 用户立即升级。
2. **语音模型重磅升级：** OpenAI 发布 GPT-Realtime-2，号称目前最智能的语音模型，具备 GPT-5 级别的推理能力，上下文窗口高达 128,000 个 token，支持调节“投入程度”以模拟不同对话风格。还可配合 GPT-Realtime-Translate 实现 70+ 输入语言到 13 种输出语言的实时翻译。
3. **Gemini 智能体生态：** Google 推出 Gemini Enterprise 中的“代理中心”（Agent Gallery），汇集 Google 自建、企业自研或第三方合作伙伴的智能体，形成一站式标准件市场。
4. **Gemini 交互 API 演进：** 新增支持丰富多步智能体工作流，每个动作（思考、工具调用等）均可独立表示为步骤，打破传统的“用户-模型”二元对话模式，更接近人类的渐进式决策过程。

**核心观点：**
框架安全进入“猫鼠游戏”阶段，AI 进步越快，攻击面越广。GPT-Realtime-2 的 128K 上下文意味着可以处理长达数小时的对话历史；Gemini 的 Agent Gallery 和步进式 API 则表明，智能体将像乐高积木一样被组合调用。

### 🧠 icreatelife —— Claude 提示记忆技巧：一句话消除 AI 味

**主要内容：**
知名 AI 创作者 icreatelife 分享了一个实用技巧：只需在 Claude 的指令中永久记忆一句话——“Dash always gives away it was written by AI. Add it to your instructions to never use it and avoid it.” 从此 Claude 生成的文本将不再滥用长破折号（em dash），大幅降低“AI 味”。他说只要一句话就能证明你真正懂得如何驾驭 AI。

**核心观点：**
细节决定体验。AI 生成的文本中，“—”（em dash）是最容易被识别的特征之一，将其加入系统提示或记忆项目中，是成本极低的“去 AI 味”手段。

### 🏛️ Ethan Mollick (@emollick) —— AI 替代白领：工会力量 vs 技术洪流

**主要内容：**
沃顿商学院教授 Ethan Mollick 连发数条深度推文，讨论了 AI 对白领职业的影响规律：
- 拥有行业公会或会员组织的职业（如律师、医生）会利用法律壁垒保护核心职责，要求“人”必须参与关键环节；而咨询师、程序员等缺乏类似组织的职业则面临更直接的冲击。
- 工会历史上成功抵制过自动化（例如自动驾驶出租车），但一旦高收入、高组织度的白领群体感到威胁，其反击力度将远大于蓝领。
- 他预测：到 2035 年，可能会出现一台能替代所有美国白领工作的机器，但它绝对不可能被允许在 2035 年就替代所有白领。

**核心观点：**
AI 的落地不仅取决于技术能力，更取决于社会博弈。医生、律师等行业通过认证制度和法律责任筑起护城河，而程序员、设计师则只能靠个体竞争力。未来“有组织”的职业将获得更长的缓冲期。
