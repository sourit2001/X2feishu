### 三、宝玉 (@dotey) — Agent vs. 传统 App + AI：执行主体的根本差异

**主要内容：** 宝玉发了一条关于 **Agent 应用**与**传统 App + AI** 的深度对比。他指出，传统模式下，人是“操作”App 的主语，AI 只是辅助工具（例如早期微软 Copilot 集成在 PowerPoint 里，只能回答问题、辅助操作）；而在 Agent 模式下，人变成了“指挥”Agent 的指挥官，Agent 自己去操作 App / CLI 完成具体任务。

**核心观点：**
- 这一区分直击 Agent 产品的设计本质：**从“人机协作”转向“人指挥机”**。
- 未来的 Agent 需要具备更强的自主规划与执行能力，而不是简单的“对话框+API”。

**实用工具/资源：**
- **Hermes Agent**（宝玉自有的 Agent 产品，他提到的 `baoyu-infographic Skill` 正是 Hermes Agent 的一部分）

**来源链接：** [推文1](https://twitter.com/dotey/status/2059063249753043358), [推文3](https://twitter.com/dotey/status/2059129670558200101)
