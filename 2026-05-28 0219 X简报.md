### 🏗️ 宝玉 (@dotey) — Agent 产品布局设计哲学

### 💰 Ethan Mollick — Token 预算灾难：企业全面告急

- **主要内容**：Ethan 连发数条推文，剖析企业 Token 预算管理危机。
  - 一年前没人把 Token 列入预算，现在已成为编码绝对必需品。
  - 大多数公司对 Token 使用只有粗浅理解，在“所有人用最多 Token”和“强制用本地模型”之间摇摆。
  - 大组织在年初几个月就烧光了全年 Token 预算。
  - 高管困惑：最好的工程师该分最多 Token 吗？最差的呢？如何避免工程师因额度不均产生怨气？
- **核心观点**：Token 经济是新资源的角力，企业急需一套成熟的分配、监控和优化体系。
- **实用工具/资源**：无具体工具，但问题本身是重要洞察。
- **来源链接**：[推文1](https://twitter.com/emollick/status/2059640930265686158)、[推文2](https://twitter.com/emollick/status/2059641771311681583)、[推文3](https://twitter.com/emollick/status/2059642485559685506)、[推文4](https://twitter.com/emollick/status/2059654566820323350)
